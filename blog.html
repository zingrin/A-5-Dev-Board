Question-1: What are the different ways to select an element in the DOM?

DOM elements by ID (getElementById), class (getElementsByClassName), tag (getElementsByTagName), or CSS selectors (querySelector,querySelectorAll).For a single element, use querySelector(), and for multiple, use querySelectorAll()!

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML – Gets or sets the HTML content inside an element. It processes and renders HTML tags. innerText – Gets or sets only visible text, ignoring hidden text. It does not process HTML tags. textContent – Gets or sets all text, including hidden text, but does not process HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation means setting an event listener on a parent element instead of each child.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event occurs on a child element, and it travels up to its parent, then grandparent, until it reaches the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element.Add it using appendChild() or prepend(). Remove it using remove().
